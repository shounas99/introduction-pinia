# PiniaApp 🍍

## Dev


This project should help you how started with Pinia in Vue.js c:

## Pasos:
### Clonar el proyecto

```sh
npm run dev
```

### 2. Ejecutar

```sh
yarn install
```
or

```sh
npm install
```

### 3.Levantar el backend server

```sh
yarn backend-server
```
or

```sh
npm run backend-server
```

### 4. Ejecutar la aplicación de Vue.js

```sh
yarn dev
```
or

```sh
npm run dev
```
